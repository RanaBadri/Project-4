White Wine Data

The data below was downloaded from UCI Machine Learning Repository. It contains data about white variants of the Portuguese "Vinho Verde" wine. Reference [Cortez et al., 2009] for more details. Due to privacy and logistic issues, only physicochemical (inputs) and sensory (the output) variables are available (e.g. there is no data about grape types, wine brand, wine selling price, etc.). This dataset can be viewed as classification or regression tasks. The classes are ordered and not balanced (e.g. there are much more average wines than excellent or poor ones represented). Outlier detection algorithms was used to detect the few excellent or poor wines. Also, our analysis has shown not all input variables has an impact or were relevant.
